Font family: Ioskeley Mono — a monospace typeface loaded at weights 300, 400, 600, and 700. Falls back to Geist Mono, SF Mono, and system monospace stacks.

Body text is set at 1rem / 16px with line-height: 1.8. The measure is capped at 68ch for comfortable reading.

Text weights in use: light 300, regular 400, semibold 600, bold 700.

Italic variants: light italic, regular italic, semibold italic, bold italic.

headings in context

h1 — page title

Used once per page as the primary title. Rendered in var(--accent) at 1.75rem, bold. Scales down to 1.5rem on mobile.

h2 — section heading

Used for major content sections. Rendered in var(--text-headings) at 1.35rem, semibold, with 2.5rem top margin.

h3 — sub-section heading

Used for sub-sections. Rendered in var(--accent) at 1.15rem, semibold.

selection & focus

Text selection uses ::selection: try highlighting this sentence — the selection background becomes var(--accent) with the page background as the text color.

Focus ring: keyboard-navigable elements receive a 2px solid var(--accent) outline at 2px offset. Mouse clicks suppress the outline via :focus:not(:focus-visible).

responsive & accessibility

breakpoints

  • ≤ 480px — mobile: reduced padding, smaller type scale, full-width pre blocks
  • ≤ 768px — tablet: slightly reduced padding and heading sizes

media query features

  • prefers-reduced-motion — all transitions/animations set to 0.01ms
  • prefers-contrast: high — higher contrast palette, underlined links, thicker code borders
  • print — white background, black text, hidden nav, absolute URLs appended to links
